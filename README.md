# NextDay.HomeTest
這是一個計算下一天的日期的程式，輸入西元 yyyy, mm, dd，則會得到下一天的日期。我們假設 DateOnly 都是合法的，不考慮錯誤輸入，也不用考慮時區問題。撰寫的時候不要使用 C# 的內建 API 進行日期和曆法的轉換。

- 加入判斷閏年規則
- 適當的重構和加入單元測試
- 如果要讓這個類別可以支援農曆，可以如何設計？

完成後的程式用 Pull Requests 的方式提交，並且在 PR 告訴我們你是誰。
